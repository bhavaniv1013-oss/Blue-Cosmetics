# Blue Cosmetics E-Commerce Store 💄

A modern, responsive e-commerce website for Blue Cosmetics built with pure HTML, CSS, and JavaScript.

## ✨ Features
- Product catalog with search & filters
- Shopping cart functionality 
- Wishlist to save favorite products
- User login/signup system
- Admin panel for managing products
- Order summary & checkout flow

## 🛠️ Tech Stack
**Frontend:** HTML5, CSS3, JavaScript ES6  
**Storage:** LocalStorage for cart/wishlist  
**Design:** Responsive UI with custom styling
